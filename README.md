# Image Manipulation Suite 🎨📸
A high-performance desktop application for real-time image processing and interactive drawing.

## 🚀 Overview
Built with **Python** and **Tkinter**, this tool provides a seamless interface for digital drawing and image editing. It utilizes the **Pillow (PIL)** library to handle complex image transformations and filtering at scale.

## ✨ Key Features
* **Interactive Canvas:** Implemented event-driven drawing logic allowing for real-time user input and brush customization.
* **Non-Destructive Filtering:** Integrated professional-grade filters including **Gaussian Blur**, **Grayscale**, and **Color Inversion**.
* **Advanced Resampling:** Utilizes **LANCZOS resampling** algorithms for high-quality image scaling and export.
* **Asset Management:** Features integrated file dialogs for importing, editing, and saving assets in multiple formats (.png, .jpg).

## 🛠️ Tech Stack
* **Language:** Python 3.x
* **GUI Framework:** Tkinter
* **Image Processing:** Pillow (PIL)
